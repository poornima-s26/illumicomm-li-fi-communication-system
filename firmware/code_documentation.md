# Firmware Source Code

This folder contains source code for the ILLUMICOMM Li-Fi Communication System.

## Code Modules

- Image and Text Transmitter
- Image and Text Receiver
- Audio Transmitter
- Audio Receiver

## Communication Methods Used

Digital Communication:

- NRZ Encoding
- ON-OFF Keying (OOK)

Analog Communication:

- PWM based optical modulation

## Programming Environment

- Arduino IDE
- ESP32 Libraries
- ESP32-CAM Libraries

The source code files are uploaded separately in this folder.
