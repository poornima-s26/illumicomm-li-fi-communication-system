## Project Status

✅ Completed Academic Project

---

## Project Overview

ILLUMICOMM is a Li-Fi based wireless communication system designed for transmitting text, image, and audio data using Visible Light Communication (VLC).

Unlike traditional RF communication, this system uses optical light intensity modulation for wireless data transmission through laser/light sources.

The project demonstrates optical communication for secure and high-speed wireless data transfer.

---

## Modules Developed

### Image and Text Transmission Module

- Text transmission using light pulses  
- Image transmission using ESP32-CAM  
- Optical modulation using laser source  

### Audio Transmission Module

- Audio transmission using analog light intensity modulation  
- Real-time signal transmission using VLC
