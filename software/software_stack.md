# Software and Programming

This project uses the following software tools.

## Programming Environment

- Arduino IDE

## Libraries Used

- ESP32-CAM Libraries
- Wi-Fi Libraries
- Serial Communication Libraries

## Algorithms Implemented

### Image Processing

- Image compression to 8×8 matrix
- Binary pixel conversion

### Text Processing

- ASCII encoding
- Binary bitstream generation

### Communication Algorithm

- ON-OFF Keying (OOK) modulation
- Digital optical communication

### Audio Processing

- Analog signal amplification
- Analog intensity modulation

## Debugging Tools

- Serial Monitor
- Web Server Interface

## Purpose

Software controls:

- Laser modulation
- Signal encoding
- Data transmission
- Receiver decoding
- Output display
