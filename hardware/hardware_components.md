# Hardware Components

This project uses the following hardware components.

## Main Controller

- ESP32-CAM Module
- ESP32 Development Board

## Transmission Section

- Laser Diode Module
- NPN Transistor Driver Circuit

## Receiver Section

- Photodiode Receiver
- LM358 Operational Amplifier
- LM393 Comparator

## Audio Section

- Microphone Module
- LM386 Audio Amplifier
- Speaker

## Power Section

- Breadboard
- Jumper Wires
- DC Power Supply

## Purpose

These hardware modules enable Li-Fi based communication for:

- Text transmission
- Image transmission
- Audio transmission
